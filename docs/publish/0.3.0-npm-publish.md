# 0.3.0 发布到 npm（作用域包）

- 包名：@ticoag/auto-commit
- Node.js：>= 22
- 发布类型：minor

## 主要变更
- 🔁 项目正式更名为 AutoCommit，默认命令更新为 `auto-commit`
- 🤖 新增 Codex provider（@openai/codex-sdk），可通过 `--provider codex` 实时切换
- 🗂️ 配置与模板目录迁移至 `~/.auto-commit/`，并自动兼容旧路径
- 📝 多语言 README / FAQ / FEATURES、安装脚本与发布指南同步更新
- ♻️ 继续保留 `claude-auto-commit` 命令作为兼容别名

## 安装与升级

一次性使用：

```bash
npx @ticoag/auto-commit@0.3.0 --help
```

全局安装：

```bash
npm i -g @ticoag/auto-commit@0.3.0
```

## 发布命令（参考）

已设置 publishConfig.access 为 public，可直接：

```bash
npm publish
```

## 注意事项
- 发布前执行 `npm whoami` 确认登录账号
- 如启用 npm 2FA，按照提示输入一次性验证码
- 推荐发布前先运行 `npm pack --dry-run` 检查打包内容
