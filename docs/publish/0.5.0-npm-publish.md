# 0.5.0 发布到 npm（作用域包）

- 包名：@ticoag/auto-commit
- Node.js：>= 22
- 发布类型：minor

## 主要变更
- **默认配置更新**：默认 Provider 调整为 `codex`，模型为 `gpt-5.1`，语言为 `zh`（中文），默认开启 Emoji。
- **移除日语支持**：移除所有日语注释与逻辑代码。
- **移除兼容性**：移除 `claude-auto-commit` 二进制别名及其兼容逻辑。

## 安装与升级

一次性运行：

```bash
npx @ticoag/auto-commit@0.5.0 --help
```

全局安装：

```bash
npm i -g @ticoag/auto-commit@0.5.0
```

## 发布命令（参考）

已设置 publishConfig.access 为 public，可直接：

```bash
npm publish
```
