# 0.4.0 发布到 npm（作用域包）

- 包名：@ticoag/auto-commit
- Node.js：>= 22
- 发布类型：minor

## 主要变更
- 新增：Codex provider（@openai/codex-sdk），支持 `--provider codex`
- 更名：CLI 命令更新为 `auto-commit`，保留旧别名
- 配置：默认目录迁移至 ~/.auto-commit，模板/配置共用
- 文档：多语言 README、发布指南、安装脚本同步更新

## 安装与升级

一次性运行：

```bash
npx @ticoag/auto-commit@0.4.0 --help
```

全局安装：

```bash
npm i -g @ticoag/auto-commit@0.4.0
```

## 发布命令（参考）

已设置 publishConfig.access 为 public，可直接：

```bash
npm publish
```

## 注意事项
- 请确认已登录 npm：
  - `npm whoami`
- 若启用 2FA，按提示输入一次性验证码
- 建议先用 `npm pack --dry-run` 预览打包内容
